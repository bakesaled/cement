# Cement


A tool that encrypts a file behind a password.

[![Build Status](https://travis-ci.org/bakesaled/cement.svg?branch=master)](https://travis-ci.org/bakesaled/cement)

## Installation

```bash
$ npm install
```

## Running the examples

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
